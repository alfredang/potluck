iOS Build Process

Learn how an iOS project is built on EAS Build. This page describes the process of building iOS projects with EAS Build.

Build Process

1 Local: Check Git Status
If cli.requireCommit is set to true in eas.json, check if the git index is clean.
2 Local: Prepare Credentials
Depending on credentialsSource, obtain credentials from local credentials.json or EAS servers.
3 Remote: Create Build Environment
Create a new macOS VM with Xcode, Fastlane, and all build tools installed.
4 Remote: Install Dependencies
Run npm install and npx expo-doctor to diagnose issues.
5 Remote: Restore Credentials
Import Distribution Certificate and Provisioning Profile into the keychain.
6 Remote: Build with Fastlane
Run pod install then fastlane gym to create the IPA.
